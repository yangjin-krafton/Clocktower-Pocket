# 밤 행동 UI 패널 정리

## 공통 2단계 흐름

모든 밤 행동은 **편집 → 공개** 두 단계로 처리한다.

```
[ 편집 단계 ]  호스트만 보는 화면
  - 자리 배치 Oval에서 공개 대상 선택/해제
  - 메시지 textarea 수정
  - [편집 완료 → 공개] 클릭
          ↓
[ 공개 단계 ]  참가자에게 보여주는 화면
  - 동일 Oval, 선택 자리만 강조 (역할 아이콘 비표시)
  - 확정된 메시지 텍스트
  - [호스트] 다음 →
```

역할별로 **편집 단계 진입 전** 선행 패널이 붙는 경우가 있다.

---

## 패널 컴포넌트

| 컴포넌트 | 역할 | 대상 |
|---|---|---|
| `HostDecisionPanel` | 정보 옵션 선택 (정확/거짓/직접 입력) | 호스트 |
| `OvalSelectPanel` | 지목 대상 자리 선택 | 참가자 테마 |
| `RevealEditPanel` | **편집 → 공개** 두 단계 통합 | 호스트 → 참가자 |
| `InfoPanel` | 단순 정보 확인 (미니언/임프 선행용) | 호스트 |
| `BluffSelectPanel` | 블러프 3개 선택 | 호스트 |
| `SpyModeSelector` | 그리모어 공개 방식 선택 | 호스트 |
| `SpyGrimoirePanel` | 그리모어 열람 | 참가자 테마 |

## 역할별 2단계 흐름

---

### minion-info — 미니언 공개
> 첫째 밤 / 미니언 전원

**선행 패널**
```
InfoPanel (호스트)
  미니언 목록 + 임프 번호 확인
  [👁 참가자에게 공개 →]
```

**편집 단계**
| 항목 | 내용 |
|---|---|
| Oval 초기 선택 | 미니언 전원 + 임프 |
| 메시지 초기값 | `임프: N번 / 동료 미니언: N번, ...` |

**공개 단계**
> 미니언 각자에게 순서대로 보여줌

---

### demon-info — 임프 정보
> 첫째 밤 / 임프

**선행 패널**
```
BluffSelectPanel (호스트)
  블러프 3개 선택
          ↓
InfoPanel (호스트)
  미니언 번호 + 블러프 역할 확인
  [👁 참가자에게 공개 →]
```

**편집 단계**
| 항목 | 내용 |
|---|---|
| Oval 초기 선택 | 미니언 자리 |
| 메시지 초기값 | `미니언: N번, ... / 블러프: 역할, 역할, 역할` |

**공개 단계**
> 임프에게 보여줌

---

### spy — 스파이
> 첫째 밤 + 매 밤 / 스파이

**선행 패널**
```
SpyModeSelector (호스트)
  공개 방식 선택: 전체 / 절반 / 셔플
```

**공개 단계** ← 편집 없이 바로
```
SpyGrimoirePanel
  전체 그리모어 열람
  [호스트] 다음 →
```

> 스파이는 RevealEditPanel 대신 SpyGrimoirePanel을 직접 사용

---

### washerwoman — 세탁부
> 첫째 밤

**선행 패널**
```
HostDecisionPanel (호스트)
  옵션 선택 →
    🔵 정확한 정보: 마을 주민 1명 + 미끼 1명
    🔴 거짓 정보 (중독 시): 무작위 쌍 + 다른 역할
    ✏️ 직접 선택: 호스트가 2명 지목
```

**편집 단계**
| 항목 | 내용 |
|---|---|
| Oval 초기 선택 | 선택한 옵션의 2명 |
| 메시지 초기값 | `세탁부 → N번, N번` |

**공개 단계**
> 세탁부에게 보여줌

---

### fortuneteller — 점쟁이
> 첫째 밤 + 매 밤

**선행 패널**
```
OvalSelectPanel (참가자 테마)
  2명 선택 (사망자 포함 가능)
          ↓
  [정상 + 은둔자/스파이 없음]      [중독 OR 은둔자/스파이 있음]
        그대로 편집 단계               ↓
                              HostDecisionPanel (호스트)
                                🔵 기본 결과
                                🔴 거짓 정보 (중독 시)
                                🧎 은둔자 → 임프로 인식
                                🕵️ 스파이 → 선으로 인식
```

**편집 단계**
| 항목 | 내용 |
|---|---|
| Oval 초기 선택 | 지목한 2자리 |
| 메시지 초기값 | `✅ 예, 그 중 한 명은 임프입니다` 또는 `❌ 아니오, 임프가 없습니다` |

**공개 단계**
> 점쟁이에게 보여줌

## 밤 순서

### 첫째 밤
```
minion-info → demon-info → poisoner → washerwoman → librarian
→ investigator → chef → empath → fortuneteller → butler → spy
```

### 둘째 밤 이후
```
poisoner → monk → imp → ravenkeeper → undertaker
→ empath → fortuneteller → butler → spy
```

## revealData 구조

```js
{
  roleIcon:    string,     // emoji 또는 'filename.png'
  roleName:    string,
  roleTeam:    string,     // 'town' | 'outside' | 'minion' | 'demon' | null
  roleAbility: string,
  message:     string,     // 메시지 초기값
  players:     [{ id }],   // Oval 초기 선택 번호
  allPlayers:  Player[],   // Oval 전체 자리 렌더링용
  action:      string,     // 행동 안내
  draft: {                 // override 초기값 (선택)
    message:  string,
    players:  [{ id }],
  },
  onNext:      Function,
}
```
