Blood on the Clocktower 게임 도우미

서버 없이 동작하는 P2P 게임 도우미

WebRTC 기반 P2P 통신을 사용하여 별도 서버 없이 플레이어들과 직접 연결합니다. 스토리텔러는 방을 만들고, 플레이어들은 코드를 입력하여 참가합니다.

👑

스토리텔러 (방장)

방을 만들고 게임을 진행합니다
🎮

플레이어 (참가자)

방 코드를 입력하여 참가합니다
프로젝트 Clocktower Pocket
타입 Static WebApp (P2P)
기술 WebRTC + Trystero
상태 P2P 연결 테스트 중

💡 사용 방법:
1. 스토리텔러가 "방장" 버튼을 클릭하여 방을 생성합니다
2. 플레이어들은 "참가자" 버튼을 클릭하고 방 코드를 입력합니다
3. 모든 참가자가 연결되면 게임을 시작할 수 있습니다
