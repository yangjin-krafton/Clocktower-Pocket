## 한눈에 보는 진행 흐름

1. 스크립트 확정 → 역할 무작위 배정
2. **첫날 밤** — 악 팀 서로 확인, 정보형 역할 첫 정보 수령
3. **낮** — 공개 토론 → [지목](day.md) → [투표](day.md) → [처형](day.md)
4. **밤** — 스크립트 밤 순서대로 각 역할 처리
5. 승리 조건 달성까지 낮/밤 반복
