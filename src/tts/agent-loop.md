# TTS 에이전트 루프 실행 규칙

이 문서는 `작가 에이전트`와 `평가 에이전트`를 빠르게 재사용하기 위한 실행 규칙이다.

## 사용 목적

- 특정 역할의 밤 TTS 샘플을 빠르게 개선한다.
- 평가 점수 `90점 이상`에 도달할 때까지 반복한다.

## 사용 파일

- 작가 프롬프트: `src/tts/agent-writer.md`
- 평가 프롬프트: `src/tts/agent-reviewer.md`
- 공통 규칙: `src/tts/작성규칙.md`

## 기본 루프

1. 대상 CSV를 정한다.
2. 작가 에이전트가 CSV를 수정한다.
3. 평가 에이전트가 점수를 매긴다.
4. 점수가 `90점 미만`이면 평가 피드백을 작가에게 전달한다.
5. 작가가 다시 수정한다.
6. 평가자가 다시 채점한다.
7. `90점 이상`이면 종료한다.

## 운영 원칙

- 한 번에 모든 것을 뜯어고치지 않는다.
- 점수에 직접 영향을 주는 문제부터 고친다.
- 보통 우선순위는 아래와 같다.

1. 길이와 정보 밀도
2. 밤 진행 방해 여부
3. 실제 단서처럼 들리는 위험
4. 캐릭터성
5. 테마 적합성

## 빠른 호출 문구 예시

아래처럼 짧게 요청하면 된다.

```text
agent-loop.md 기준으로 세탁부 샘플 다시 돌려줘
```

```text
작성규칙 기준으로 조사관 샘플 만들고 agent loop 90점까지 돌려줘
```

```text
사서 csv에 작가/평가 에이전트 루프 실행
```

## 종료 조건

- 평가 점수 `90점 이상`
- 또는 사용자가 중단 지시
- 또는 더 이상 의미 있는 개선이 없다고 합의된 경우
