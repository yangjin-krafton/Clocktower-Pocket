# v0.2 업데이트

## 일정 조율 캘린더
- 랜딩 페이지에 **2주 캘린더** 추가
- 참가 가능한 날짜를 탭하여 등록
- 다른 유저의 등록 현황을 실시간 확인
- 이름과 선택 날짜는 자동 저장되어 재접속 시 복원

## 역할 계승 시스템
- **임프 자살 계승**: 임프가 스스로를 선택하면 다른 미니언에게 악마 역할 이전
- **진홍의 여인 계승**: 악마 사망 시 진홍의 여인이 즉시 악마로 승격
- 밤 페이즈 중 즉시 교체 적용

## 역할 아이콘 리뉴얼
- 모든 역할 아이콘을 고해상도 공식 에셋으로 교체
- 상태 아이콘 (독, 보호, 취함 등) 업데이트

## 야간 행동 개선
- **장의사** 처형 대상 추적 기능 추가
- **시장** 바운스 처리 (시장 처형 시 다른 플레이어 사망)
- **남작** 슬롯 경고 및 인원 분배 테이블 개선
- 야간 힌트 및 어드바이저 UI 개선

## 블러프 & 메모
- 블러프 선택 패널에서 기존 블러프 미리 선택 표시
- 메모 기능 개선 및 플레이스홀더 업데이트

## 여행자 역할 카드 추가
- **희생양**, **총잡이**, **거지**, **관료**, **도둑** 5종 추가
- 역할 도감에서 아이콘, 능력 설명, 규칙 페이지 확인 가능
- 게임 내 기능은 아직 미구현 (도감 열람 전용)

## 기타 개선
- 방 코드 공유 시 링크 복사 기능 추가
- 슬롯 마크 시스템에 독약 효과 및 집사 마스터 표시 추가
- 전반적인 UI/UX 개선 및 버그 수정
