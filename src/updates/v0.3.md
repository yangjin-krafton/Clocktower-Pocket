# v0.3 업데이트

## 여행자 인게임 구현
- **여행자 추가/관리**: 호스트와 참여자 모두 자리배치 탭에서 여행자 등록 가능
- **위치 선택**: 원탁 기반 "N번 — M번 사이" 위치 지정 (기존 여행자 사이에도 삽입 가능)
- **진영 선택 (호스트)**: 여행자 추가 시 선/악 진영 결정, 슬롯 테두리 색으로 표시
- 로비(게임 시작 전)부터 게임 도중까지 언제든 여행자 추가 가능

## 여행자 밤 행동
- **관료**: 매 밤 1명 선택 — 다음 낮 투표 3표로 계산
- **도둑**: 매 밤 1명 선택 — 다음 낮 투표 음수로 계산
- 엔진 밤 순서 완료 후 자동으로 여행자 밤 스텝 진행
- 그리모아 낮 화면에 투표 수정 알림 카드 표시

## 여행자 낮 행동
- **총잡이**: 투표 중 인접 플레이어 즉사 선택
- **희생양**: 처형 확정 시 대체 처형 여부 팝업 선택
- **추방 (Exile)**: 모든 여행자에 대해 추방 투표 가능 (처형 횟수 소모 안 함)
- **거지**: 투표 토큰 메커닉은 현장 진행 (도감 참조)

## 밤 행동 UI 개선
- 능력 설명 핵심 용어 자동 염색 (마을 주민, 미니언, 숫자 등)
- **정보 역할 선택 검증**: 세탁부/사서/조사관이 잘못된 자리를 선택하면 경고 팝업 표시
  - 예: "선택한 2명 중 마을 주민이 없습니다" → 다시 선택 또는 이대로 진행

## 기타
